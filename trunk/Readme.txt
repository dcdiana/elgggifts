Installation:

1. Copy the gifts plugin to you mod folder
2. Enable the gifts plugin in your "Tool Administrator"
3. Select the amount of gifts you want to provide in the admin menu (save your settings)
4. Define your gifts on the "Giftssettings"-Tab

If you are using a version < 0.1.0 and uploaded some pictures, you have to change the pictures
admin menu. There is a new HTML-Box for each Gift, where you can link any type of media (e.g. YouTube Videos,
Pictures or just some Text)
